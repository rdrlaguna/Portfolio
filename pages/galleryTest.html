MARIO

Pyramid generator

This program recreates a pyramid using hases(#) for bricks.

Introduce a number between 1 and 8 for the pyramid's height and click the RUN button to see your pyramid.

CASH

Coin counter

This program prints the minimum coins needed to make the given amount of change in cents.

Introduce a whole number greater than zero. Recall that the number cannot be a decimal, since we are counting cents.

CREDIT

Card validator

This program checks the validity of a given credit card number.

Introduce a credit card number and press the RUN button to check whether if it is a valid American Express, MasterCard, or Visa card number.

CAESAR

Text cipher

This program enables you to encrypt messages using Caesar's cipher.

Introduce a whole number followed by the text you want to cipher. The number would be the key the program will use to cipher your text. Separate the number and the text using a space: key text.
